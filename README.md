# Kuvaa Store

Premium fashion e-commerce platform built with React + Vite + Supabase.

## Setup

1. Run `supabase-setup.sql` in your Supabase SQL Editor
2. Create `.env` file with your Supabase credentials (see `.env.example`)
3. Deploy to Vercel and add environment variables

## Admin Access

Go to `/admin/login` — create a user in Supabase Auth dashboard first.

## Tech Stack

- React + Vite
- Supabase (database + auth + storage)
- React Router
- React Hot Toast
- Lucide React icons
